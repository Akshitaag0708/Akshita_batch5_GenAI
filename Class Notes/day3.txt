slicing :
used to extract a part of collection datatype 
syntax: var[start:end:step]
end is always excluded
+/- 1 is passed with end depending on condition
left to right then +1 and right to left -1
if u want to go till end of collection then u can skip end value and same for start
s[2: :1]//till end
s[ :4:1]//from start
if step value is 1 then u can skip it also 
-1 step for reverse
s[: :-1]
