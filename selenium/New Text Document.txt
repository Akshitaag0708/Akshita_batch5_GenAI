 driver=Chrome(options=o)    // if we do not pass options=o then also browser will blink

->># driver.close()      # suppose we open more tab manually in browser then this will close the url tab only as the sleep time completes  

->>driver.quit()        # suppose we open more tab manually in browser then this will close all tabs including url page as the sleep time completes  

NOTE-- for single tab both close() and quit() will perform samely 
NOTE-- but for many tabs close() will close only the tab that driver is carrying whereas quit will close the whole browser means all tabs get closed
